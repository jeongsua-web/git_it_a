### git 명령어
- git init (.git 생성)
- git remote add origin 레포주소 (Github 원격 연결)
- git remote -v (연결된 원격 저장소 확인)
- git add . (깃폴더에 있는 모든것 추가!)
- git commit -m '커밋메세지' (""큰따옴표 오류 나서 작은따옴표로 함. 작은따옴표가 오류가 적대)
- git push -u origin 브랜치이름 (다음부터는 그냥 git push 만 써도 됨)
- git checkout 브랜치이름 (브랜치 변경)
