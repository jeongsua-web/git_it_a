## Git 명령어

* `git init` → `.git` 생성
* `git remote add origin <레포주소>` → Github 원격 연결
* `git remote -v` → 연결된 원격 저장소 확인
* `git add .` → 모든 파일 스테이징
* `git commit -m '커밋메세지'` → 커밋

  * 큰따옴표 오류가 날 때는 작은따옴표 사용
* `git push -u origin <브랜치이름>` → 최초 푸시

  * 다음부터는 그냥 `git push` 사용 가능
* `git checkout <브랜치이름>` → 브랜치 변경

## Firebase

* FlutterFire CLI
  Flutter 앱에서 Firebase 서비스(Firestore, Auth, Storage 등)를 쉽게 연동하도록 도와주는 도구

```bash
flutterfire --version       # 설치 여부 확인
dart pub global activate flutterfire_cli   # CLI 설치
flutterfire configure       # Firebase 프로젝트 연결 및 설정
```
